### 1. Node 설치
   - https://nodejs.org
   - LTS 버전 설치

### 2. 폴더 만들기
```
mkdir chatter
cd chatter
```

### 3. 만든 폴더에 채팅 프로그램 다운로드 (오른클릭 > 저장 > 만든 폴더 선택)
   - https://raw.githubusercontent.com/jiwonpark/tts-mqtt-master/master/chatter.js

### 4. 명령창에서 만든 폴더 들어가서 필요한 node 패키지 (mqtt) 설치
```
npm install --save mqtt
```

### 5. 채팅 실행
```
node chatter
```
